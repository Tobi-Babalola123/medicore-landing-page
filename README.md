# Medicore Landing Page Template

A modern and responsive Medicore landing page template built with React and Tailwind CSS. Designed for startups, businesses, and digital products.

## Features

- Fully responsive design
- Built with React + Tailwind CSS
- Clean and modern UI
- Easy to customize
- SEO friendly structure

## Sections Included

- Hero Section
- Features Section
- Product/Service Showcase
- Testimonials
- Pricing
- Call to Action
- Footer
