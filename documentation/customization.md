# Customization Guide

This template is designed to be easy to customize.

## Changing Text Content

Most text content can be found inside the components located in:

src/components

Simply open the component and edit the text.

## Changing Images

Images are stored in:

public/images

Replace the images with your own while keeping the same file names.

## Updating Colors

Colors are controlled using Tailwind CSS classes.

You can modify colors in the components or update the Tailwind configuration file.

## Updating Sections

Each section of the landing page is separated into components.

Example:

Hero Section
Features Section
Testimonials Section

You can edit, remove, or rearrange these components as needed.

## Deployment

The template can be deployed to platforms such as:

- Netlify
- Vercel
- GitHub Pages
